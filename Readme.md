# Arch Linux setup

## Pakete

#### sddm

This is the [display manager](https://wiki.archlinux.org/title/Display_manager). 
A display manager is providing the login screen and is responsible for launching the selected desktop environment after login. 
Without a display manager installed, a tty would be shown at the end of the boot process.
